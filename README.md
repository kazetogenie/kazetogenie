## Concepts

[カゼトゲタナゴ](https://en.wikipedia.org/wiki/Kyushu_bitterling)から着想を得たもの。当時、日本淡水魚を飼育しておりウェブで見たカゼトゲタナゴの姿と言葉の美しさにあやかってというところです。

* Kaze (Wind): 風を起こす、風に乗る、風を操る、台風の目になる、停滞を吹き飛ばす、風に溶け込み何気ない日常の風景となる
* Toge (Achieve) : トゲ→遂げる、踏破する、困難を突き破る、研げ→研鑽する
* Genie: 魔人→精霊→魔法のランプ→なんでもかなう→実現する、ゲニ→銭→正当正道かつ王道の価値を生み続ける

呼称はカゼトゲーニ、カゼトジーニです。不死鳥のタマゴトジーニもいいですね。

## どこへ行こうというのかね？

活動予定はとあるソフトウェアの長期開発とアクアリウム関連の執筆くらいですね。翻訳もできますが精力的には活動しないです。

1. ソフトウェア開発で使う処理系の資料整備
2. メダカなど日本淡水魚に関する飼育解説書の執筆
3. とあるソフトウェアの長期開発

これらが活動初期の方向性です。
